=====================
Distributed Consensus
=====================






A distributed consensus algorithm

Usage
--------

You should install pipenv package in python enviroment before run the code, then run 'pipenv install' to configure this project enviroment.
Then change the path 'D:\sjtu-distributed-consensus' in 'config.sceneIII.12xdual.bat' into your actual path of project, and run 'config.sceneIII.12xdual.bat'.
You can observe information from command window and cache file.
